Guild Operating Vows

Local First

Models between 1B and 7B parameters run on community-owned machines for everyday guidance.

Consent Aware

Conversations are opt-in for archiving; private reflections stay on the visitor's device.

Escalation with Accountability

Complex queries require a quick steward review before using shared cloud credits.

Control Console Snapshot

  • Switch between ultra-fast (1B), balanced (3B), and high-quality (7B) local models.
  • Toggle cloud assistance for research-heavy requests with transparent logging.
  • View live stats: tokens per second, memory footprint, and response latency.
  • Deploy new language packs and cultural modules curated by the community.

Forge Queue

These are high-priority tasks for the Companion Artificers:

  • Offline knowledge packs for the Living Library so companions can quote sources even without connectivity.
  • Shared prompt notebooks, allowing visitors to gift well-crafted prompts to one another.
  • Mutual aid dispatcher that routes requests from the Steward Board into the AI companion workflow.
  • Accessibility tuning with voice-first interactions and captioned audio responses.
